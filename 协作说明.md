# Git 协作说明

本文件夹已作为 Git 仓库使用，远程仓库：

```text
https://github.com/aikenchen0-ctrl/superAppAndroid.git
```

## 日常使用

开始编辑前先同步：

```bash
git pull
```

编辑完成后提交并推送：

```bash
git status
git add .
git commit -m "说明本次修改内容"
git push
```

其他电脑首次使用：

```bash
git clone https://github.com/aikenchen0-ctrl/superAppAndroid.git
```

## 注意

- 不提交 `build/`、`.gradle/`、`*.apk`、`*.zip` 等构建产物和大文件。
- 多人同时修改同一个文件时，先 `git pull` 再编辑，减少冲突。
- 如果出现冲突，先确认保留哪一版内容，再提交解决后的文件。
