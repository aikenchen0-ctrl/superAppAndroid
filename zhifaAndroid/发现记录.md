# 发现记录

- 底部条触摸窗口的宽高当前写死为 156dp x 30dp，视觉线条为 92dp，底边距为 0dp。
- 底部条是屏幕底部居中的 `TYPE_ACCESSIBILITY_OVERLAY`，不覆盖展开聊天页中上部的两侧头像。
- 展开聊天时服务仅停止原生侧边手势；已创建的回退 `EdgeOverlayView` 没有移除，可能继续覆盖两侧头像。
- 现有 `GestureAction`、动作选择器和侧边手势偏好可复用。
- OnePlus 会优先处理全高度系统返回手势；要在展开聊天页可靠捕获整侧手势，调试包需要 ADB 授予 `WRITE_SECURE_SETTINGS`，并将原生触摸目标限制为 4dp 到 12dp。
